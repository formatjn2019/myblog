2335.装满杯子需要的最短总时长（简单）

[2335.装满杯子需要的最短总时长（简单）](https://leetcode.cn/problems/minimum-amount-of-time-to-fill-cups/)



解法1：模拟


每次倒所需最多的两杯

```go
func fillCups(amount []int) int {
    sort.Ints(amount)
    result := 0
    for amount[1] > 0 {
        amount[2] -= 1
        amount[1] -= 1
        result++
        sort.Ints(amount)
    }
    result += amount[2]
    return result
}
```



解法2：贪心

根据所需最大杯数和其余杯数推理即可

```go
func fillCups(amount []int) int {
	sort.Ints(amount)
	if amount[2] >= amount[1]+amount[0] {
		return amount[2]
	} else {
		return (amount[0] + amount[1] + amount[2] + 1) / 2
	}
}
```
