1470.重新排列数组
[1470.重新排列数组](https://leetcode.cn/problems/shuffle-the-array/)



看了题解和提示，没有能原地计算模拟后坐标的方法，都是时换空的解法。

那么不如直接模拟好了

#### 解法1：模拟

```go
func shuffle(nums []int, n int) []int {
	result := make([]int, 2*n)
	for i := 0; i < 2*n; i += 2 {
		result[i] = nums[i/2]
		result[i+1] = nums[i/2+n]
	}
	return result
}

```
