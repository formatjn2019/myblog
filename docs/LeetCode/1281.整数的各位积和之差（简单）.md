1281.整数的各位积和之差（简单）

[1281.整数的各位积和之差（简单）](https://leetcode.cn/problems/subtract-the-product-and-sum-of-digits-of-an-integer/description/)



解法：模拟



```go
func subtractProductAndSum(n int) int {
	sum, prod := 0, 1
	for t := n; t > 0; t /= 10 {
		sum += t % 10
		prod *= t % 10
	}
	return prod - sum
}
```
