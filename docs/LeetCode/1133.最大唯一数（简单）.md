1133.最大唯一数（简单）

[1133.最大唯一数（简单）](https://leetcode.cn/problems/largest-unique-number/)



解法：哈希



```go
func largestUniqueNumber(nums []int) int {
	result := -1
	arrayCount := make([]int, 1001)
	for _, num := range nums {
		arrayCount[num]++
	}
	for i := len(arrayCount) - 1; i >= 0; i-- {
		if arrayCount[i] == 1 {
			result = i
			break
		}
	}
	return result
}
```
