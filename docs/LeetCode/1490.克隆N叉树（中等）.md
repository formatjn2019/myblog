1490.克隆N叉树（中等）

[1490.克隆N叉树（中等）](https://leetcode.cn/problems/clone-n-ary-tree/)



解法：DFS递归拷贝



经典递归树问题



```go
func cloneTree(root *Node) *Node {
	if root == nil {
		return nil
	}
	var children []*Node
	for _, child := range root.Children {
		children = append(children, cloneTree(child))
	}
	return &Node{Val: root.Val, Children: children}
}
```
