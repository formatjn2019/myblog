0507.完美数

[0507.完美数
](https://leetcode-cn.com/problems/perfect-number/)

简单题不需要题解吧

```java
public boolean checkPerfectNumber(int num) {
    return num >= 2 && getFactorSum(num) == num;
}

private int getFactorSum(int num) {
    int result = 1;
    for (int t = 2; t < Math.sqrt(num); t++) {
        if (num % t == 0) {
            result += t;
            if (num / t != t) {
                result += num / t;
            }
        }
    }
    return result;
}
```
