面试题05.02.二进制数转字符串（中等）

[面试题05.02.二进制数转字符串（中等）](https://leetcode.cn/problems/bianry-number-to-string-lcci/)



解法：枚举

依次乘二获取整数位即可，若字符串大于32返回error

```go
func printBin(num float64) string {
	var bf bytes.Buffer
	bf.WriteByte('0')
	bf.WriteByte('.')
	for bf.Len() < 34 && num != 0 {
		num *= 2
		if num >= 1 {
			num -= 1
			bf.WriteByte('1')
		} else {
			bf.WriteByte('0')
		}
	}
	if bf.Len() > 32 {
		return "ERROR"
	}
	return bf.String()
}
```
