1614.括号的最大嵌套深度



解法：一次遍历

```java
public int maxDepth(String s) {
    int result = 0;
    int depth = 0;
    for (char c : s.toCharArray()) {
        if (c == '(') {
            depth++;
            result = Math.max(result, depth);
        } else if (c == ')') {
            depth--;
        }
    }
    return result;
}
```
