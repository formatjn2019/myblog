2520.统计能整除数字的位数（简单）

[2520.统计能整除数字的位数（简单）](https://leetcode.cn/problems/count-the-digits-that-divide-a-number/)



解法：模拟



依次对各个位上的数字进行取模计算即可



```go
func countDigits(num int) int {
	var result int
	for tn := num; tn > 0; tn /= 10 {
		if tn%10 != 0 && num%(tn%10) == 0 {
			result++
		}
	}
	return result
}
```
