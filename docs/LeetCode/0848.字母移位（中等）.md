0848.字母移位
[0848.字母移位](https://leetcode-cn.com/problems/shifting-letters/)

这道题应该划到简单的


解法：倒序遍历

```go
func shiftingLetters(s string, shifts []int) string {
	chars:=[]rune(s)
	for sum,i:=0,len(shifts)-1;i>=0;i-- {
		sum=(sum+shifts[i])%26
		chars[i]='a'+(chars[i]-'a'+rune(sum))%26
	}
	return string(chars)
}
```
