0141.环形链表

[0141.环形链表
](https://leetcode-cn.com/problems/linked-list-cycle/)

解法：双指针，快慢指针

经典老题了，如果新题还不一定会写



```java
public boolean hasCycle(ListNode head) {
    //快慢指针
    for (ListNode slow = head, fast = head; slow != null; slow = slow.next) {
        fast = fast == null ? fast : fast.next;
        fast = fast == null ? fast : fast.next;
        if (fast == slow) {
            return true;
        }
    }
    return false;
}
```
