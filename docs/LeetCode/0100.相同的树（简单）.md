0100.相同的树

https://leetcode-cn.com/problems/same-tree/

解法：递归



```java
public boolean isSameTree(TreeNode p, TreeNode q) {
    if (p == q) {
        return true;
    }
    if (p == null || q == null || p.val != q.val) {
        return false;
    }
    return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
}
```
