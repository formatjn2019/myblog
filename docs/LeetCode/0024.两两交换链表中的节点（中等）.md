0024.两两交换链表中的节点

[0024.两两交换链表中的节点
](https://leetcode-cn.com/problems/swap-nodes-in-pairs/)

递归调用，一次后移两个单位，获取下个位置的指针

```java
public ListNode swapPairs(ListNode head){
    if (head==null || head.next==null){
        return head;
    }
    ListNode second= head.next;
    head.next=swapPairs(second.next);
    second.next= head;
    return second;
}
```
