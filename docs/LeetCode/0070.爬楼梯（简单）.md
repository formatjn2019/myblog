0070.爬楼梯

[0070.爬楼梯
](https://leetcode-cn.com/problems/climbing-stairs/)

解法： 动态规划

习惯用数组表示。。。

java

```java
public int climbStairs(int n) {
    int[] judgeArray = new int[]{1, 2, 0};
    for (int i = 2; i < n; i++) {
        judgeArray[i % 3] = judgeArray[(i + 1) % 3] + judgeArray[(i + 2) % 3];
    }
    return judgeArray[(n - 1) % 3];
}
```
