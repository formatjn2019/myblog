0089.格雷编码

[0089.格雷编码
](https://leetcode-cn.com/problems/gray-code/)

格式

```
G(0) = 0b000 = 0 ==> 0 ^ (0>>1) = 0

G(1) = 0b001 = 1 ==> 1 ^ (1>>1) = 1

G(2) = 0b011 = 3 ==> 2 ^ (2>>1) = 3

G(3) = 0b010 = 2 ==> 3 ^ (3>>1) = 2

G(4) = 0b110 = 6 ==> 4 ^ (4>>1) = 6

G(5) = 0b111 = 7 ==> 5 ^ (5>>1) = 7

G(6) = 0b101 = 5 ==> 6 ^ (6>>1) = 5

G(7) = 0b100 = 4 ==> 7 ^ (7>>1) = 4
```



公式 `i^i>>1`

```java
public List<Integer> grayCode(int n) {
    List<Integer> result = new LinkedList<>();
    for (int i = 0; i < Math.pow(2, n); i++) {
        result.add(i ^ i >> 1);
    }
    return result;
}
```
