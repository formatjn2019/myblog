2395.和相等的子数组（简单）

[2395.和相等的子数组（简单）](https://leetcode.cn/problems/find-subarrays-with-equal-sum/)



解法：哈希表

```go
func findSubarrays(nums []int) bool {
	sumMap:=map[int]bool{}
	for i:=1;i<len(nums);i++{
		if _,ok:=sumMap[nums[i]+nums[i-1]];ok{
			return true
		}
		sumMap[nums[i]+nums[i-1]]=true
	}
	return false
}
```
