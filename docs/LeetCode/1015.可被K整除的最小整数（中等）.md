1015.可被K整除的最小整数（中等）

[1015.可被K整除的最小整数（中等）](https://leetcode.cn/problems/smallest-integer-divisible-by-k/)



解法：数学

```go
func smallestRepunitDivByK(k int) int {
	if k%2 == 0 || k%5 == 0 {
		return -1
	}
	result, remain := 1, 1
	for ; remain != 0 && result < k; result++ {
		remain = (remain*10 + 1) % k
	}
	return result
}
```
